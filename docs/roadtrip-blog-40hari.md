## 🛠️ Cara per Hari

```
1. Buka /admin/prompt-generator
2. Pilih provinsi + kota (dari jadwal)
3. Klik "✨ Generate dengan DeepSeek"
4. Preview → Klik "Import JSON"
5. Spot + Roadtrip terbuat otomatis ✅

6. Buka /admin/blog/new
7. Klik "💡 Ide Konten" → pilih ide
8. AI generate → Edit → Publish ✅
```

**Total waktu: ~1 jam/hari.**
